using CSS

definition

when you write pure html code the browser uses it's default boring styling, and here is where CSS is usfull.
CSS stands for "Cascading Style Sheets" it is the language used to style your web page
it has all the defrent features like adding colors, change writing syles, controling text position and alot more.

linking CSS with html

to apply CSS to your html file, the most used way is to creat a css file and link it with your html file using the link element
and set the "href" attribute to the path of your css file then add a rel attribute (realtionship) and set it to "stylesheet"
make sure the new link element is located within a head element
here is the syntax :

<link rel="stylesheet" href="style.css">

 or alternatively for quick styling without having to creat a new file you can use the html style element here is the syntax:

<style>
  h1{
      color: blue;
  }
</style>

 the syntax inside the style element is the same syntax used inside a CSS file wich is explained next.

basic CSS syntax

lets say we want to style one of the HTML elements. here is the syntax:

selector {
    property=valeu;
}

 the selector is mainly used to chose the html element we want to apply the changes on it
the property is used to specify the thing you want to chage (color, size, font ,etc...)
and the valeu depends on the property, for example if the property is set to color
the valeu should be a color (rgb(231, 6, 6) or just red) and if the property is a font-family
the value should be the desired font name (Arial (sans-serif), Verdana) etc...

some css properties

in CSS there is a large collection of properties and each property can have defrent type of valeu
depending on the property itself but we are only going to cover a minimal amount of them
here is a full list : full list link

  • color

    • this property is strait forward, it allows you to change the color of the specified selector.
    you can give it the color name directly like this: "color: red".
    you can also use the rgb foramation of the color you want like this : "color: rgb(100, 0, 204);"

  • background

    • there are several properties to manipulate the appearance of the element background:

    1. background color

      2. if you want to change the background color of the selector you can do it like this:
      selector {
          background-color="red";
      }

    2. background Transparency

      3. this property is only usfulle when the background-color is also used so that you can midify
      the Transparency of the background of an element using this syntax:

      selector {
          background-color="red";
          opacity: 0.5;";
      }

       the default opacity valeu is 1 so that means the examle abouve will reduce the Transparency of
      the red background by 50%.

    3. image as a background

      4. to show an image as the background of some element we can use the "background-image" property like this:
      selector {
          background-color="red";
          opacity: 0.5;";
      }

more CSS selector usages

  • multiple selectors

    • the selector part of the CSS syntax can be used for more than just selecting one html element there are some additional features
    for example you can applly the same changes to a group of elements by setting the selector to all the elements like this:

    h2, h3, h4 {
        background-color: black;
    }

     this will set the background color of all the specified elements (h2, h3, h4) to black color.
    this syntax allows you to apply the same styling to multiple of elements without
    having to write each one individually.

  • using class as selector

    • class is an html atribute that can be shared acros multiple elements here is how we can benifit from it in CSS :

    .class_value{
        font-family: 'Times New Roman', Times, serif;
    }

     this syntax abouve is used to make all the elements with an atribute class of value "class_value" follow the specified style changes
    in our case the changes will apply to every element no matter what type as long as its class atribute is set to the same value used as the selector .
    you can also use it in that way:

    p.class_val{
        text-align: center;
    }

     the defrence here is that we specified both of the html element and class we want to target
    wich means only the <p> elements that have a valeu of "class_val" will be aligned to the center

  • uinvesal selector

    • sometimes you want to make some changes on all of you html elements without exepetions here is how:

    *{
        text-align: center;
    }

     when a '*' character is used as the selector, the styling will be applied to all of the elements.

  • id as a selector

    • the id is an html attribute the defrence between an id and a class is that each element can have a unique id
    wich means if we use as out CSS selector it will be applied to the element with that same id here is it's syntax:

    #id-valeu{
        color: red;
    }
