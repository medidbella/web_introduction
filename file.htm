html elements

while reading html files you will encounter allot of elements that begins
with an opening tag "<>" and a closing tag "</>"

  • titles element

    • title element can be used once in your file to give your page a name like the one
    obove that says "web introduction" the syntax to use that element :"<title > your title </title>" .

  • paragraphs

    • paragraph element are used to insert text to your page and this current text is written
    using this element .the syntax to use that element : "<p> your paragraph </p>" .

  • headings

    • as the name says this element is used to add headings to your web page there are up to
    6 hedings as the number increment the size of the heding decrement.
    the syntax to use that element :"<h1..6> </h1..6 >."

  • unordered lists

    • unordered lists are used to create bulleted lists such as the one that i used in this page to type the
    elements (paragraphs headings ...).
    the syntax to use that element : "<ul> <li> <li>... <ul>".
    inside <ul> element (unordered list) you can add your <li> elements here is an examle:

    <ul>
    <li> part 1 </li>
    <li> part 2 </li>
    </ul>

    this will make a bulleted list of two parts (part 1 and part 2).

  • ordered lists

    • ordered lists are the same as unordered lists, exesept that instead of using points
    to mark the nodes of the list, it uses number in assending order.
    the syntax to use that element : "<ol> <li> <li>... <ol>".

using CSS

CSS stands for "Cascading Style Sheets"it is the language used to style your web page by adding colors change writing syles
and label themes ext...

set up

to set up and start using CSS, you should first creat a css file and link it with your html file using the link element
and set the "href" attribute to the path of your css file thrn add a rel attribute (realtionship) and set it to "stylesheet"

basic CSS syntax

lets say we want to style the HTML paragraph element here is the syntax:

	selector {
		property=valeu
	}

the selector is used to chose the html element we want to apply the changes on it
the property is used to specify the thing you want to chage (color, size, font ,etc...)
and the valeu depends on the property, for example if the property is set to color
the valeu should be a color (rgb(231, 6, 6) or just red) and if the property is a font
the value should be the desired font name (Arial (sans-serif), Verdana) etc...
